# User Posting system using MERN stack

Users can login to that portal and post something about the industry. Other users can view the posts and the comments on the same.
The author of the post can comment back or anyone can comment on the comment.

## Some important commands

1. Go to directory `client`, and run `npm install`
2. Go to directory `server`, and run `npm install`
3. In one terminal, run `mongod`
4. In `server` directory, run `npm run dev`
5. In `client` directory, run `npm run start`
